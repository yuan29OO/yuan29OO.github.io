User Interface Modeling of Android Mobile Apps for Software Optimization
Machine Learning Team Project
Modeled how the users interact with different UIs and predict the users’ next actions.
  Explored RICO datasets which contained UI layout vectors and interaction traces data.
  Retrieved similar App UI using nearest neighbor search and encoded UI vector with activity data.
  Trained with RNN models. Tuned and evaluated the models.

Optimization of MVICFG generated from Hydrogen
Team Project
  Hydrogen can generate MVICFG on top of different versions of source files. However, it is hard to read and understand the MVICFG since the nodes are represented in LLVM instructions and the graph is too big to zoom in to see the details. Our project aims to simplify MVICFG to a smaller sized but more interpretable graph to developers. The result showed that our simplified MVICFG is able to greatly reduce the number of nodes of MVICFG.
